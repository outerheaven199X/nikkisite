ABOUT THIS BUILD: DATABOX

ABOUT THIS BUILD

DATABOX is a RAG (Retrieval-Augmented Generation) system that demonstrates AI security vulnerabilities through prompt injection attacks. Built with Node.js, SQLite FTS5, and multiple LLM adapters, it provides a safe environment for learning about AI security. The system includes comprehensive audit logging and injection detection to show both offensive and defensive AI security techniques.

BUILD STATUS: Production-ready, actively maintained
PURPOSE: AI security education and prompt injection research
